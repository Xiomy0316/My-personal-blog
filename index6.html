Tutorial de Git

Descripción

Aprende paso a paso cómo usar la terminal de Git para trabajar en tu proyecto.

¿Qué es Git?

Git es un sistema de control de versiones distribuido, lo que significa que un clon local del proyecto es un repositorio de control de versiones completo. Estos repositorios locales plenamente funcionales permiten trabajar sin conexión o de forma remota con facilidad.

En un sistema de control de versiones distribuido, cada desarrollador tiene una copia integral del proyecto y del historial del mismo, proporcionándoles una vista consistente y unificada a los miembros del equipo de desarrollo.
